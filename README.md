# O-Ring-Gland-Design-Document
A quantitative, engineering-grade technical reference for O-ring gland design across multiple seal applications. This repository contains interpolated design data, parametric selection charts, and worked examples for standard metric cross-sections (1-6 mm), compiled from industry handbooks and validated against ISO 3601/DIN 3771 standards.
